# Data Dictionary

This file explains the fields used in the incident dataset for the Strategic IT Operations Dashboard.

| Field | Description |
|---|---|
| Ticket ID | Unique identifier for each incident ticket |
| Open Date | Date when the incident was opened |
| Close Date | Date when the incident was closed |
| Priority | Incident priority level such as P1, P2, P3, or P4 |
| Support Team | Team assigned to resolve the incident |
| Status | Current or final ticket status |
| SLA Status | Indicates whether the ticket met or breached SLA |
| Resolution Time | Time required to resolve the incident |
| Root Cause | Main cause related to the incident or SLA breach |
| Category | Type of incident or service affected |
